# 1 安装城市设计平台

# 1.1 安装 UDSetup.exe 软件

安装完成后先不要启动/运行软件，如果已经启动/运行，请先关闭。

# 1.2 解压数据至单独文件夹

# 2 安装超图客户端

# 2.1 解压超图客户端软件

不需要安装，解压后即可使用，但尚未获得软件运行许可。

# 2.2 运行超图客户端软件包

找到已解压的超图客户端软件包，找到\Supermap 10i\Tools\SuperMapLicenseCenter\SuperMap.LicenseCenter(for .NET 4.0).exe，双击运行（首次运行需要一些时间）。

# 2.3 核对许可状态

运行完成后，若 Supermap.LicenseCenter(for .NET 4.0).exe 软件关闭，请再次打开。核对许可状态下是否有试用许可。如果有，则已经获得超图软件的许可证书，可以正常运行超图软件。若无，以管理员身份运行该软件，再次尝试。

# 2.4 运行超图软件

双击运行超图软件，初次打开可能需要一些时间，请耐心等候。

# 3 配置数据

# 3.1 修改数据本地路径

找到城市设计 UD 平台安装位置，找到\UD\AppConfig.json，以管理员身份打开进行编辑。

确认 Appconfig 中 IsLocal 为 true。

找到 1.2 步解压的数据位置，按住 shift 右键点击威海城市设计.smwu，点击复制为路径。将路径粘贴至 Appconfig 中（框出来的第二行）。将路径中\改成\，保存。

# 3.2 打开文件型工作空间

在超图软件中，未命名空间右键，选择打开文件型工作空间。

# 3.3 选择 smwu 文件

请务必先确保本机网络畅通，找到城市设计 UD 平台安装位置，找到\Data\威海城市设计.smwu，选择打开威海城市设计.smwu。

# 3.4 成功打开 smwu 文件

初次打开数据可能较慢，请耐心等候，数据打开成功如下：

# 3.5 删除场景

在场景-威海城市设计右键，删除威海城市设计场景。

# 3.6 打开 iServer 服务场景

在场景右键，选择打开 iServer 服务场景

# 3.7 输入服务地址和场景名称

服务地址: http://52.83.134.236:8090/iserver/services/3D-WeiHaiChengShiSheJi-CS/rest/realspace 场景名称:威海城市设计

点击确定，开始连接服务场景。初次连接可能较慢，请确保内网畅通，稍等片刻。

# 3.8 再次打开场景

服务场景打开后，双击场景威海城市设计，打开场景，初次打开，可能较慢，请稍等片刻。

# 3.9 定位到当前威海市范围

打开场景后，请双击影像图图层，定位到当前威海市范围。

# 3.10 确定好显示范围

确定好显示范围，并务必确保影像图和 DEM 图层处于显示状态，并关闭其他图层的显示。

# 3.11 保存场景

在当前场景，按下 Ctrl+S，保存当前场景。

# 3.12 关闭超图客户端软件

关闭超图客户端软件，如弹出保存警告，请再次点击保存。

# 4 安装成功，运行城市设计平台

# 4.1 打开威海城市设计 UD 软件

双击运行威海城市设计 UD 软件，登录账号后，即可正常使用软件。
