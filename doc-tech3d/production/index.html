# 快速入门

三维 GIS，细分为 CS、BS 两个方向。

CS 端产品包括：UD 三维空间智能管控和UD 三维报建系统。

UD 三维空间智能管控和UD 三维报建系统均基于SuperMap iObjects .Net 10i开发的以WPF为界面框架的数字化平台。

其中UD 三维空间智能管控承载着城市设计转型与提升的重要使命，弥补城市设计到实施管控转译过程中出现的问题，将城市设计对城市空间品质提升的各项要求落实到空间管控的层面。城市设计数字化转型，实现城市设计“从三维空间到三维数字化空间”、“从设计到规划管理全环节”。

BS 端产品包括：sm3d-core(三维内核库)、三维空间智能管控平台、三维一张图系统。

BS 端产品均基于Vue框架，以SuperMap iClient3D for WebGL为三维引擎开发的系统平台。
