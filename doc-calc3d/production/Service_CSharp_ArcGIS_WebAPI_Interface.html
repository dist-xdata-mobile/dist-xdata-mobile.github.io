# ***1.2******获取合规审查规则信息接口***

接口名称 getPlanCheckRuleInfo
接口描述 获取合规性审查的规则信息

详细说明 根据参数项目类别（xmlb）匹配 PlanCheckRule.xml 文件里的“RuleGroup“的“name”属性获得下面的“Rule”节点的规则信息
涉及配置 PlanCheckRule.xml（合规性审查规则配置文件）
