# 部署手册—数据更新管理系统

作者:Dist 时间： 2020-10-16

# 1. 文档介绍

# 1.1. 文档目的

为了使实施人员更快的部署“数据更新管理系统”，尽量缩短部署时间，特此编写此部署手册。本手册结合实际操作，配以大量的文字说明和图片展示，确保高效快捷的部署本系统。

# 1.2. 读者对象

本文档的预期读者对象包括：

（1）数据更新管理系统相关用户；

（2）项目组成员，包括项目经理、设计人员、开发人员、测试人员等。

# 2. 系统环境

本节描述系统运行所需的软硬件及网络环境。本系统要求环境为工具的最佳适应环境，在要求环境下保障工具的正常运行和服务支持；其他环境下可能出现难以预判的工具问题，只能提供有限的服务支持，无法保障解决所有问题

# 2.3. 网络环境

网络带宽主干网为 100MB，桌面带宽不小于 10MB。

# 3. 部署内容

# 3.1. 数据库

数据库 MDMJN 为数据更新管理系统使用的数据库，用于存储数据更新系统的配置。

# 3.2. 创建数据库

利用 toad 或者 PLSQL，创建用户名为 MDMJN，密码为 pass，表空间为 MDMJN 的数据库；

CREATE TABLESPACE MDMJN DATAFILE 'E:\app\Administrator\oradata\orcl\MDMJN.DBF' SIZE 100M AUTOEXTEND ON NEXT 50M MAXSIZE UNLIMITED;

CREATE USER MDMJN

IDENTIFIED BY MDMJN -- 为该用户指定登录密码

DEFAULT TABLESPACE MDMJN -- 为该用户指定表空间

TEMPORARY TABLESPACE TEMP

PROFILE DEFAULT

ACCOUNT UNLOCK;

-- 2 Roles for XXGIS 为该用户指定角色

GRANT CONNECT TO MDMJN;

GRANT RESOURCE TO MDMJN;

GRANT DBA TO MDMJN;

ALTER USER MDMJN DEFAULT ROLE ALL;

-- 1 System Privilege for XXGIS 为该用户指定权限

GRANT UNLIMITED TABLESPACE TO MDMJN;

GRANT SELECT ANY SEQUENCE TO MDMJN;

GRANT CREATE ANY TABLE TO MDMJN;

GRANT CREATE VIEW TO MDMJN;

GRANT DROP ANY VIEW TO MDMJN;

GRANT EXECUTE ANY PROCEDURE TO MDMJN;

GRANT SELECT ANY TABLE TO MDMJN;

GRANT UPDATE ANY TABLE TO MDMJN;

-- 1 Tablespace Quota for XXGIS

ALTER USER MDMJN QUOTA UNLIMITED ON MDMJN;

# 3.3. 导入数据库

打开 CMD 窗口，输入命令。

命令格式如下：（下划线部分需根据实际替换）

imp 用户/密码@IP 地址:端口号/数据库实例名 file=文件路径 full=y

如：imp MDMJN/pass@172.16.100.94/ghdb file=c:\MDMJN.dmp full=y

# 3.4. 业务库表数据清理

序号 表名 描述
1 DCP_GIS_TASKEXTENT 存放任务更新范围信息
2 DCP_HISTORYDBPAIR 存放历史库配置信息
3 DCP_INITEDLAYER 存放已初始化图层信息
4 DCP_SCHEME 存放方案名称信息
5 DCP_SYSCONFIG 存放数据库配置名称信息
6 DCP_SYSCONFIG_PARA 存放数据库配置信息
7 DCP_TASK 存放任务名称信息
8 DCP_TASK_LAYERINFO 存放任务图层信息
